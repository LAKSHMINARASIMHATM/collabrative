# Terminal Keyboard Shortcuts

## ✅ Implemented Shortcuts

### Navigation
- **←** (Left Arrow) - Move cursor left
- **→** (Right Arrow) - Move cursor right
- **↑** (Up Arrow) - Previous command in history
- **↓** (Down Arrow) - Next command in history
- **Home** - Move cursor to start of line
- **End** - Move cursor to end of line
- **Ctrl+A** - Move to start of line
- **Ctrl+E** - Move to end of line

### Editing
- **Backspace** - Delete character before cursor
- **Delete** - Delete character at cursor
- **Tab** - Auto-complete commands
- **Ctrl+V** - Paste from clipboard
- **Ctrl+U** - Clear line before cursor
- **Ctrl+K** - Clear line after cursor

### Control
- **Enter** - Execute command
- **Ctrl+C** - Cancel current command
- **Ctrl+L** - Clear screen

### Selection & Copy
- **Mouse Selection** - Select text with mouse
- **Copy Button** - Click copy button in header
- **Right Click** - Context menu (browser default)

## Usage Examples

```powershell
# Navigate through command history
↑ ↓

# Edit command in middle
npm install react
←←←←←← (move to 'install')
Backspace Backspace... (delete 'install')
run dev (type new command)

# Paste from clipboard
Ctrl+V

# Clear parts of line
Ctrl+U (clear before cursor)
Ctrl+K (clear after cursor)
```
